# Android / Termux deployment

## Language

- English: [Android / Termux deployment](android-termux.md)
- Chinese: [Android / Termux 部署](../../deployment/android-termux.md)

## Positioning

This is an experimental / fallback option, not the preferred production setup.

## Watch out for

- battery optimization may kill the background process
- network switching can affect reachability
- heat and battery wear matter
- pair it with Cloudflare Tunnel if you need public access

## Annual switching

Prefer changing external config or env vars; do not reinstall the app.
